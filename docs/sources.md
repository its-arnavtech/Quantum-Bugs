# Sources

This repository is based on quantum mutation testing tools, real quantum bug benchmarks, and empirical quantum software bug studies.

## Mutation Testing Tools

### Muskit

Muskit is a mutation analysis tool for Qiskit quantum programs. It supports add gate, remove gate, and replace gate mutation operators.

Repository/paper link to add:

* [https://github.com/Simula-COMPLEX/muskit](https://github.com/Simula-COMPLEX/muskit)

### QMutPy

QMutPy is a mutation testing tool for Qiskit quantum algorithms and applications. It extends classical Python mutation testing with quantum-specific operators for gates and measurements.

Repository/paper link to add:

* [https://github.com/jose/qmutpy](https://github.com/jose/qmutpy)

### QCRMut

QCRMut is a random mutant generator for Qiskit quantum circuits. It supports insertion, deletion, gate name modification, and gate attribute modification.

Paper link to add:

* [https://arxiv.org/abs/2410.01415](https://arxiv.org/abs/2410.01415)

### QuanForge

QuanForge is a mutation testing framework for quantum neural networks. It supports gate-level and parameter-level mutation operators.

Paper link to add:

* [https://arxiv.org/html/2604.20706v1](https://arxiv.org/html/2604.20706v1)

### QMutBench

QMutBench is a static benchmark of quantum circuit mutants generated using mutation operators.

Paper link to add:

* [https://arxiv.org/html/2604.15870v1](https://arxiv.org/html/2604.15870v1)

## Real Bug Datasets and Studies

### Bugs4Q

Bugs4Q is a benchmark of real Qiskit bugs with buggy and fixed versions.

Repository link to add:

* [https://github.com/Z-928/Bugs4Q](https://github.com/Z-928/Bugs4Q)

### Qiskit Bug Pattern Research

This research identifies recurring Qiskit bug patterns such as unequal classical bits and qubits, repeated measurement, custom gate recognition issues, and unsafe uncomputation.

### Empirical Quantum Platform Bug Study

This empirical study analyzes real bugs across multiple quantum computing platforms and identifies quantum-specific and classical software bug classes.

Paper link to add:

* [https://software-lab.org/publications/quantum_bugs_arXiv_2110.14560.pdf](https://software-lab.org/publications/quantum_bugs_arXiv_2110.14560.pdf)
