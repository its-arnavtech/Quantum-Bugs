# Example Bug Record

## Bug Class

Wrong Target Qubit

## Bug Class ID

QB-004

## Category

Gate-level

## Description

A quantum gate is applied to the wrong qubit or qubit pair.

## Why This Matters

Quantum circuits are sensitive to qubit position and entanglement structure. Applying the right gate to the wrong qubit can produce an entirely different output distribution.
